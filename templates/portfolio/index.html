Predicting working memory failure using machine learning

In this project I worked with a time series dataset; EEG data recorded during a memory task experiment. The goal of the project was to use machine learning to predict when the subject was about to forget the memory item. My approach led us to be able to predict working memory failure with 78% accuracy.
